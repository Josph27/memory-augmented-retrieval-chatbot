# src/core/ — Core Data Contracts

**TL;DR:** `contracts.py` defines all frozen dataclasses that flow through the agent pipeline.
`MemoryCandidate` normalizes every memory source into one shape. `ContextPacket` is the single
authoritative context assembly the answer model receives. `WorkflowTrace` captures the full turn
trace from routing through answer.

---

## Reference

### `MemorySourceType`

```python
MemorySourceType = Literal[
    "recent_messages", "structured_memory", "document_memory",
    "current_chat_gist", "current_chat_span",
    "previous_chat_gist", "raw_message_span",
    # Backward-compatible aliases:
    "current_chat_chunks", "previous_chat_memory",
    "short_term", "long_term", "document", "raw_messages",
    "unknown",
]
```

**Purpose:** Enum-like literal for all retrievable memory sources.
Legacy aliases (`current_chat_chunks`, `previous_chat_memory`, `short_term`, `long_term`,
`document`, `raw_messages`) exist for backward compatibility. New code should use the
non-aliased values.

**Used by:** Every retriever, `RoutePlan`, `MemoryCandidate`, `RetrieverDispatcher`.

---

### `SourcePlan`

```python
@dataclass(frozen=True)
class SourcePlan:
    source: MemorySourceType
    enabled: bool = True
    reason: str | None = None       # Why this source was selected
    query: str | None = None        # Retrieval query for this source
    limit: int | None = None        # Max candidates from this source
    filters: dict[str, Any] = {}    # Source-specific filters
```

**Purpose:** A planned retrieval for one memory source. Created by the routing layer and
consumed by `RetrieverDispatcher`. One `RoutePlan` contains multiple `SourcePlan` entries.

**Used by:** `RoutePlanner`, `RoutingAgent`, `SemanticRouter` produce it.
`RetrieverDispatcher` consumes it to invoke individual retrievers.

**Pipeline position:** Between routing and retrieval.

---

### `RoutePlan`

```python
@dataclass(frozen=True)
class RoutePlan:
    query: str
    sources: list[SourcePlan] = []           # Which sources to query
    intent: str | None = None               # Classified user intent
    confidence: float | None = None         # Routing confidence [0,1]
    requires_retrieval: bool | None = None  # Whether retrieval is needed
    ranking_profile: str | None = None      # Ranking strategy key
    context_profile: str | None = None      # Context budget/selection key
    fallback_policy: str | None = None      # What to do if no results
    update_policy: str | None = None        # Memory update behaviour
    termination_policy: str | None = None   # When to stop the turn
    metadata: dict[str, Any] = {}
```

**Purpose:** The full routing decision for one user turn. Produced by the routing
layer and consumed by the coordinator, retrievers, reranker, and context manager.

**Context profile values** in the codebase:

| Profile | Effect |
|---|---|
| `general_chat` | Default; balanced budget, recent messages dominate |
| `memory_recall` | Working memory capped to ~8K tokens |
| `document_question` | Document retrieval enabled, Chroma scope set |
| `mixed_memory_document` | Balanced split between memory and document sources |
| `global_summary` | Large budget (up to 128K), chronological coverage |
| `structured_memory_plus_recent_messages` | Pairs structured memories with recent messages only |

**Pipeline position:** Between routing and coordinator.

### `MemoryCandidate`

```python
@dataclass(frozen=True)
class MemoryCandidate:
    source: MemorySourceType
    content: str
    score: float | None = None
    record_id: str | int | None = None       # DB row or Chroma chunk ID
    chat_id: str | None = None               # Originating chat
    source_message_ids: list[int] = []       # Raw message IDs for provenance
    metadata: dict[str, Any] = {}
```

**Purpose:** The unified shape for every retrieved context item, regardless of source.
All retrievers emit `MemoryCandidate` objects. The reranker consumes and reorders them.
The context selector trims them into a `ContextPacket`.

**Pipeline position:** Between retrieval and context selection.

---

### `ContextBudget`

```python
@dataclass(frozen=True)
class ContextBudget:
    max_tokens: int | None = None
    system_tokens: int | None = None
    memory_tokens: int | None = None
    recent_message_tokens: int | None = None
    retrieval_tokens: int | None = None
    reserved_response_tokens: int | None = None
    source_token_budgets: dict[str, int] = {}  # Per-source allocations
    metadata: dict[str, Any] = {}
```

**Purpose:** Token budget allocation for context construction.
`max_tokens` is the total context window available. The other fields break it down by
section. `source_token_budgets` maps source names to individual limits.

**Used by:** `ContextBudgetAllocator` produces it. `ContextBuilder` and
`EvidenceConstrainedContextSelector` consume it.

**Pipeline position:** Between budget planning and context assembly.

---

### `ContextPacket`

```python
@dataclass(frozen=True)
class ContextPacket:
    chat_id: str
    system_prompt: str | None = None
    structured_memory: str | None = None
    recent_message_ids: list[int] = []
    candidates: list[MemoryCandidate] = []   # Selected, ranked candidates
    budget: ContextBudget | None = None
    model_messages: list[dict[str, str]] = []  # Final model-ready messages
    metadata: dict[str, Any] = {}
```

**Purpose:** The single authoritative context assembly passed to the answer model.
`model_messages` is the final list of role/content dictionaries ready for the LLM API.
`candidates` holds the evidence items selected after reranking and budget-aware trimming.

**Pipeline position:** Between context selection and answer generation.

---

### `WorkflowTrace`

```python
@dataclass(frozen=True)
class WorkflowTrace:
    trace_id: str
    chat_id: str
    route_plan: RoutePlan | None = None
    retrieved_candidates: list[MemoryCandidate] = []  # Pre-reranking
    ranked_candidates: list[MemoryCandidate] = []     # Post-reranking
    context_budget: ContextBudget | None = None
    context_packet: ContextPacket | None = None
    termination_reason: str | None = None
    errors: list[str] = []
    metadata: dict[str, Any] = {}
```

**Purpose:** Full turn trace capturing every decision point:
the route plan, raw retrieved candidates, reranked order, budget, final context packet,
any errors, and the termination reason.

**Used by:** `CoordinatorAgent` populates it. `AnswerInspector` reads it for
observability. `MemoryTrace` stores a serialized version.

**Pipeline position:** Accumulated across the entire turn.

---

### `OrchestrationResult`

```python
@dataclass(frozen=True)
class OrchestrationResult:
    context_packet: ContextPacket
    trace: WorkflowTrace
    mode: str                       # "native", "langgraph_shadow", "langgraph_demo"
    fallback_used: bool = False
    error: str | None = None
```

**Purpose:** Wraps a `ContextPacket` with its trace and the orchestration mode that
produced it. The coordinator uses this to route into the answer agent, with fallback
awareness if the primary orchestration path failed.

**Used by:** `CoordinatorAgent`, `orchestration/` modules.

**Pipeline position:** Between context building and answer generation.

---

### `AgentTurnResult`

```python
@dataclass(frozen=True)
class AgentTurnResult:
    answer: str
    chat_id: str
    trace_id: str
    termination_reason: str
    trace: WorkflowTrace
    assistant_message_id: int | None = None
    metadata: dict[str, Any] = {}
```

**Purpose:** The final result of one user turn through the entire agent pipeline.
Bundles the answer string, trace, and persistence IDs for downstream consumers.

**Used by:** `CoordinatorAgent` returns it. `ChainlitDataLayer` stores it.
`ShortTermMemoryAgent` reads it for memory update scheduling.

**Pipeline position:** Terminal output of the turn.

---

## Data Flow

```text
RoutePlan
  └─ SourcePlan[]          ← routing layer
       ↓
  MemoryCandidate[]        ← retrievers (dispatcher)
       ↓
  MemoryCandidate[]        ← reranker (reordered)
       ↓
  ContextBudget            ← budget allocator
       ↓
  ContextPacket            ← context selector + builder
       ↓
  AgentTurnResult          ← answer agent + coordinator
       ↑
  WorkflowTrace accumulates across all steps
```

Every stage appends to `WorkflowTrace`, producing a complete provenance record.
All types are frozen — they cannot be mutated after construction.
