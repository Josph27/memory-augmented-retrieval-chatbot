# Memory-Augmented Retrieval Chatbot

A multi-agent typed-memory RAG chatbot. Ask questions, upload documents,
and the system retrieves from recent conversation, cross-chat structured
memory, and uploaded documents.

- **New Chat**: send a message to start.
- **Upload**: attach a document and ask about it in the same turn.
- **Inspect**: click "Inspect answer" on any response to see retrieval provenance.
- **End Chat**: finalizes memory extraction and gist generation for the session.
